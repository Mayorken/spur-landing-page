# 🚀 Deploy to Netlify in 60 Seconds

**No downloading. No setup. Just click and deploy.**

---

## ⚡ Fastest Way (1 Click)

### Drag & Drop to Netlify

1. Open: https://app.netlify.com/drop

2. Select the entire `landing` folder from your computer

3. Drop it on the page

4. ✅ **Done!** Your site is live in seconds

Get URL like: `https://your-site-name.netlify.app`

---

## 📋 Alternative: GitHub Integration (2 minutes)

If your code is on GitHub:

1. Push the `landing` folder to GitHub
   ```bash
   git add landing/
   git commit -m "Add landing page"
   git push
   ```

2. Go to: https://app.netlify.com

3. Click **"New site from Git"**

4. Select your GitHub repo

5. Set **publish directory** to: `landing`

6. Click **Deploy**

7. ✅ **Live in < 1 minute**

---

## 🎯 Update Your Page

After deployment, edit `landing/index.html`:

**Change these:**
- `sales@spurapp.dev` → Your email
- `$999` → Your price
- Any copy/text

Then redeploy:
```bash
git add landing/index.html
git commit -m "Update landing page"
git push
```

Netlify auto-redeploys in seconds.

## 🌍 Get Custom Domain

1. Buy domain: `spurapp.com` (Namecheap, GoDaddy, etc.)
2. In Netlify: Site settings → Domain management
3. Add your domain
4. Follow DNS instructions
5. ✅ Your domain is live with HTTPS!
